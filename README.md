# Network+ Study Notes

Personal study notes for the CompTIA Network+ (N10-009) certification.
Target exam date: June 2026.

These notes are written in my own words as I work through the official 
exam objectives, Professor Messer's video course, and Jason Dion's 
practice materials. Organized by exam domain for easy reference.

## Structure

Notes are organized by the five N10-009 exam domains:

- **01 — Networking Concepts** (23%): OSI model, TCP/IP, ports & protocols, cabling, IP addressing, subnetting
- **02 — Network Implementation** (20%): Routing, switching, wireless, physical installations
- **03 — Network Operations** (19%): Monitoring, documentation, DR, policies
- **04 — Network Security** (14%): Security concepts, hardening, physical security
- **05 — Network Troubleshooting** (24%): Methodology, tools, common issues
